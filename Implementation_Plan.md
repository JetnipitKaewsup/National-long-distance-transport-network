# Implementation Plan (4 Weeks)

### Week 1: Research & Requirement Analysis
- Study basic concepts:
  - Network latency
  - QoS
  - Wireless vs wired networks
- Define system requirements
- Create architecture diagrams

Deliverables:
- Problem statement
- System requirements
- Initial architecture design

---

### Week 2: Network Design & Simulation
- Design network topology
- Simulate networks using tools such as:
  - Packet Tracer / Mininet / NS-3 (conceptual)
- Compare:
  - Terrestrial vs satellite latency

Deliverables:
- Network topology diagram
- Simulation results

---

### Week 3: Emergency Network Design
- Design drone mesh network (conceptual)
- Study ad-hoc and mesh routing
- Analyze strengths and limitations

Deliverables:
- Emergency network design
- Performance comparison table

---

### Week 4: Final Integration & Presentation
- Integrate all layers into one system design
- Prepare final report
- Prepare presentation slides

Deliverables:
- Final report
- Presentation
