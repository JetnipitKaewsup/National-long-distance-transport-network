✅ Preparation Checklist

  • ทดสอบเปิดไฟล์ HTML ทั้ง 3 ไฟล์ (index.html, ilnp-demo.html, drone-demo.html)
  • ตรวจสอบว่า animation ทำงานปกติ
  • ซ้อมการเปลี่ยนระหว่าง demo แต่ละไฟล์
  • เตรียม laptop + projector + remote clicker
  • จับเวลา 14.5 นาที
  • ทีมซ้อมพร้อมกันอย่างน้อย 2 รอบ
  • เตรียม backup (ไฟล์ PDF, offline mode)
  • ตรวจสอบเสียงและไมโครโฟน

💡 Quick Tips for Presentation

🎯 Key Messages

  • ILNP ลด handover จาก 150ms → 15ms
  • Triple-Layer ครอบคลุมทุกสถานการณ์
  • Edge AI ป้องกันก่อนเกิดเหตุ
  • Drone mesh สร้างเครือข่ายใน 3 นาที

⚠️ Common Mistakes

  • พูดเกินเวลา → ดูนาฬิกาตลอด
  • demo ไม่ทำงาน → เตรียม screenshot สำรอง
  • สมาชิกพูดแทรกกัน → ซ้อมให้ดี
