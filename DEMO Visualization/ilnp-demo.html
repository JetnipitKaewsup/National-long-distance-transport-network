📘 ILNP Working Principle

1. Normal Operation (5G Zone)

Identifier: TRUCK-6733
Locator: 5G.CELL.342
→ Vehicle connected via 5G

2. Handover Trigger

Signal Strength: -85 dBm
Threshold: -80 dBm
→ Initiate handover to satellite

3. After Handover

Identifier: TRUCK-6733
Locator: SAT.BEAM.012
→ Same ID, new location

✨ Key Advantage: Locator Registry

เมื่อ Locator เปลี่ยน ระบบจะอัปเดตที่ Registry ทันที:

LocatorRegistry.update({
  identifier: "TRUCK-6733",
  old_locator: "5G.CELL.342",
  new_locator: "SAT.BEAM.012",
  timestamp: "09:47:23.456"
});
